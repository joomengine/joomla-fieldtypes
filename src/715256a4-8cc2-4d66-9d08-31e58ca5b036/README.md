### JCB! Field Type
# Modal Menu

> Styled dropdown of menu items.

The Modal menu form field type provides a styled drop-down list of Joomla menu items, visually matching Joomla's modal UI patterns. Unlike the modal select field type, it does not open a modal window for selection but instead presents menu items in a compact, dropdown format consistent with the modal design language used in Joomla's administrator interface.
