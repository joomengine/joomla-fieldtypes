### JCB! Field Type
# Hidden

> Hidden field for storing fixed values.

The hidden form field type provides a hidden field for saving a field whose value cannot be altered directly by a user in the Administrator (it can be altered in code or by editing the params.ini file). If the parameter has a saved value, this is entered; otherwise, the default value (if defined) will be used.

| Property | Example | Adjustable | Mandatory | Description |
|----------|---------|------------|-----------|-------------|
| type | hidden | ![no](https://img.shields.io/badge/no-blue?style=flat-square) | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | (mandatory) must be hidden. |
| name | mysecretvariable | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | (mandatory) is the unique name of the field. |
| default | 1 | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | (mandatory) is the data which needs to be collected. |
| filter | STRING | ![yes](https://img.shields.io/badge/yes-success?style=flat-square) | ![no](https://img.shields.io/badge/no-blue?style=flat-square) | (optional) allow the system to save certain html tags or raw data. |
